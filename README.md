# Codeclause_Unit-Converter

This project includes converting standard units of measurement into required units.</br>
The types of measurements used here are </br>
•	Length</br>
•	Weight</br>
•	Temperature</br>

Tools Used:</br>
IDE: VS Code</br>

Front-End languages:</br>
•	HTML</br>
•	CSS</br>
•	JavaScript</br>
